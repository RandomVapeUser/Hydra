# Hydra

The goal of the project is test discords features with no current goal in mind.

## Installation

```
# Cloning the repo
git clone https://github.com/RandomVapeUser/Hydra.git
cd Hydra

# Dependencies
pip install -r requirements.txt

# Run Hydra
python main.py
```
